Elegant pill button — use for every call to action; `primary` (coral) is the default, `gold` for premium/booking emphasis, `nude`/`outline`/`ghost` for secondary actions.

```jsx
<Button variant="primary" size="lg">Agendar agora</Button>
<Button variant="gold">Baixar catálogo</Button>
<Button variant="outline" size="sm">Ver técnicas</Button>
```

Variants: `primary | gold | nude | outline | ghost`. Sizes: `sm | md | lg`. Props: `full`, `icon`, `iconRight`, `as="a"` for links. Always uppercase, tracked, pill-shaped.
