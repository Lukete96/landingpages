A single price-list line — service name with a dotted leader to the price, plus an optional maintenance sub-line. Recreates the "Valores" catalog page.

```jsx
<PriceRow name="Volume Brasileiro" price="R$110" sub="manutenção" subPrice="R$90" />
<PriceRow name="Limpeza de Pele" price="R$79,90" />
```

Props: `name`, `sub`, `price`, `subPrice`.
