The catalog's core unit — a white card with a photo, uppercase service name, short description and price. Lifts gently on hover. Use in the Procedimentos / services grid.

```jsx
<ServiceCard
  image="assets/procedimentos.jpeg"
  name="Extensão de Cílios"
  description="Volume e curvatura sob medida para o seu olhar."
  price="R$110"
  priceNote="a partir de"
  badge="Mais procurado"
/>
```

Props: `image`, `name`, `description`, `price`, `priceNote`, `badge`, `imageHeight`, `onSelect`.
